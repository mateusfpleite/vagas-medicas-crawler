## Legenda Técnica

| Sigla | Significado | Impacto |
|-------|------------|--------|
| F | FÁCIL | 8-12h dev; baixa manutenção |
| M | MÉDIO | 1-2 semanas dev; manutenção média |
| D | DIFÍCIL | 2-4 semanas dev; manutenção alta |
| HTML | Complexidade estrutura | Simples = Regex OK; Dinâmico = CSS selectors; SPA = Rendering obrigatório |
| Login | Requer autenticação | Bloqueia crawling massivo |
| JS | JavaScript pesado | Requer Selenium/Playwright ou Splash |
| CAPTCHA | Proteção anti-bot | Requer CAPTCHA solver (custo) |
| Rate-Limit | Proteção de taxa | Requer delays + proxy rotation |
| Proxy Needed | Necessidade IP Rotation | Requer BrightData/similar (~$50-200/mês) |
